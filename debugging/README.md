# ChatGPT Introduction – Debugging

## Description
This project focuses on using ChatGPT to identify, fix, and document errors in different code samples. The tasks include debugging Python scripts, fixing HTML/JavaScript issues, and documenting existing code.

## Learning Objectives
- Identify syntax and logical errors in code
- Debug Python scripts
- Fix issues in HTML and JavaScript
- Understand and document existing programs

## Files

- `factorial.py` – Fix an infinite loop in a factorial calculation script.
- `print_arguments.py` – Print command-line arguments passed to the script.
- `change_background.html` – Fix a JavaScript error preventing a button from changing the background color.
- `mines.py` – Debug a simple Minesweeper game implemented in Python.
- `factorial_recursive.py` – Add documentation to a recursive factorial implementation.
- `checkbook.py` – Document a simple checkbook system.
- `tic.py` – Identify and fix logical errors in a Tic-Tac-Toe game.
